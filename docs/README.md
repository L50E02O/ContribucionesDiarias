## ¿Qué es este proyecto?

Sistema automatizado que mantiene una racha de contribuciones en GitHub mediante:
- Commits automáticos diarios
- Pull Requests automáticos (opcional)
- Ejecución 24/7 en Railway (gratis)
- Configuración flexible y personalizable

## Inicio Rápido

Si es tu primera vez, comienza con:
1. Lee [QUICK_START.md](QUICK_START.md)
2. Sigue [RAILWAY_DEPLOY.md](RAILWAY_DEPLOY.md)
3. Configura según [CONFIGURATION.md](CONFIGURATION.md)

## Características Principales

- **Gratis**: Funciona en el plan gratuito de Railway
- **Automático**: Commits diarios sin intervención manual
- **Flexible**: Configura número de commits, horarios, mensajes
- **Seguro**: Código abierto, sin acceso a tus datos
- **Fácil**: Setup en menos de 10 minutos

## Soporte

Si tienes problemas:
1. Revisa [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
2. Verifica los logs en Railway
3. Abre un issue en el repositorio
